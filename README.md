# More Toggleable Items 

Allows Toy Robots, Teeth, and Clocks to be turned off with Item Primary Use. Clocks now also have a 5% chance to be turned off by default, and whether they're active will be saved to your save file.

## Mod Incompatibilities 
- Any mod that adds Primary Item Use functions to Toy Robots, Teeth, or Clocks 
- Any mod that adds save data associated with individual Clock items
